Parent Questionnaires

…Could Do Better

Support for ‘middle groups’ in each class. Through the extra support that we give to all of our classes all children benefit. When small groups of children work outside the classroom the class becomes smaller, therefore giving class teachers more time with each pupil. Twice weekly each class has an extra teacher working with them to ensure more individual attention for all children.
